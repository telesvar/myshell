# myshell

A simple, embeddable POSIX shell.

## Status

__Work-in-progress__. Things break and don't work.

Currently myshell offers:
- Job system
- Builtin and external command execution
- I/O redirections
- Pipes
- If, for, while statements
- Basic command substitution

## Build

```sh
make
```
